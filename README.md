# What is this?
**OCR-систему нового поколения для обработки банковских документов (чеки, договора, выписки), которая работает лучше Tesseract.**

## Dependencies
*to install, run:*
* create virtual env
```
python3 -m venv .venv
```
* to activate env
```
source .venv/bin/activate
```
* finally to install
```
pip install -r requirements.txt 
```
* to run
```
streamlit run app.py
```
